Svbstrate

v0.9.0

Svbstrate is a base set of atomic styles suitable for any project. What is here is easily configurable to suit your needs, and since it's built on SASS, it's easy to extend and build on.

Scale

The type scale is very integral to the underlying uniformity of whitespace in this system. Elements influenced by typographical elements are scaled and spaced based on either rem or em according to the font size applied. By default, this includes all typography, lists, buttons, tables, and forms.

With a base font size – 62.5% (10px) by default – relative units serve well as units of whitespace in most projects, all the while maintaining the uniformity of space we see in the type scale.

Whitespace

Svbstrate includes whitespace classes out of the box, but these should be configured or removed on a per project basis. They use a syntax first developed by Tachyons and BASSCSS:

Prefix Description
m margin
p padding
t top
b bottom
l left
r right
v vertical
x top, bottom, left, right
n negative
Suffix Scale
0 0 (reset)
025 0.25em
05 0.5em
075 0.75em
1 1em
2 2em
4 4em
6 6em
8 8em

The prefixes and suffixes above combine to define classes like these:

.mx-0 {
  margin: 0; 
}
.mhn-2 {
  margin-left: -2em; 
  margin-right: -2em; 
}

pv-4 {
  padding-top: 4em;
  padding-bottom: 4em;
}
.pt-075 {
  margin-top: 0.75em;  
}

Typography

Type is set in rem units and receives no default margin or padding. The type scale is easily applied via the .f1 .f2 .f3 .f4 .f5 .f6 and .p classes. To add whitespace, wrap your content in a .text-content class.

Buttons

Buttons are sized using em, so creating different sized buttons is as easy as applying a different type scale class.



<button class="h2 uppercase mv-05">Click Me</button>
<button class="h4 uppercase mv-05">Click Me</button>
<button class="h6 uppercase mv-05">Click Me</button>
